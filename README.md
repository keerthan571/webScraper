# Web Scraper (Java + Maven)

A Java-based web scraping application that extracts product information 
(Name, Price, Rating) from an e-commerce website and stores it in CSV format.

This project uses Maven for dependency management and JSoup for HTML parsing.

---

## Tech Stack

- Java 22
- Maven
- JSoup
- File Handling (CSV)

---

## Features

- Multi-page web scraping
- Extracts product name, price, and rating
- Stores data in structured CSV file
- Console logging for scraping progress
- Timestamp-based output file generation

---## How to Run

1. Navigate to project root (where `pom.xml` exists)

2. Compile the project

3. Run the application
